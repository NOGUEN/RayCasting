# cub3D

NOT THAT PERFECT!

I just made it alone to understand raycasting.

You can run this only on MacOS.

## install and run

```shell
## download
git clone git@github.com:CrazyImSoFlutter/cub3D.git

## compile
make

## run
./cub3d map/wp_map.cub
```
